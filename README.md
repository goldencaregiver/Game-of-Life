# Game of Life

Консольная реализация "Игры Жизнь" Джона Конвея на языке C с использованием библиотеки ncurses (для динамических приколов). Данный проект был реализован группой студентов в рамках интенсива от Школы21 (Сбербанк).

## Особенности

- Поле 80×25 клеток
- Управление клавишами:
  - `Пробел` - выход из игры
  - `A` - увеличить скорость (уменьшить задержку)
  - `Z` - уменьшить скорость (увеличить задержку)
- Эффект "тора" — клетки видят друг друга сквозь границы
- Загрузка начальных карт из файлов
- Сохранение финальной карты

## Файлы карт

Программа ожидает файлы:

- map_1.txt ... map_5.txt — стандартные карты
- saved_map.txt — сохраненная карта
Формат файла: числа 0 или 1 через пробел, 25 строк по 80 чисел.

В репозитории уже предзагружена карта map_example.txt, в ней содержатся ТОЛЬКО нули. Её можно отредактировать как хочется и переименовать, соответственно, в нужны вам map_n.txt :)
